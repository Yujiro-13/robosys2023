# robosys2023
robosys 2023 assignment

ロボットシステム学 2023年度の授業課題用リポジトリです。

## サポートする環境
- Ubuntu22.04LTS
- wsl2
- python3.7~3.10

## インストール方法
- "git"を使用して本パッケージをダウンロードします。
```
$ git clone https://github.com/Yujiro-13/robosys2023.git
```

## 実行方法
- 例 plus
```
$ seq 5 | ./plus
```
- 結果
```
$ 15
```
